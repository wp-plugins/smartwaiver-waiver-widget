== Description ==

Allows inclusion of Smartwaiver Waiver Widget into WordPress site.

== Installation ==

1. Upload the `smartwaiver` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure Plugin via Settings -> Smartwaiver page.
